# mackolik-futbol-analiz-platformu-75
Futbol maçlarını analiz eden ve kullanıcıların maç stratejileri, oyuncu performansları ve istatistikleri hakkında değerlendirmeler yapmalarını sağlayan bir web uygulaması. Modern UI/UX prensipleriyle tasarlanmış olup, Vite, React, TypeScript, Tailwind CSS, Lucide React ve Framer Motion kullanılarak geliştirilecektir. 8px grid spacing, Glassmorph...
